CameraControlInterface Toolbox

Use your reflex camera from Matlab

Project maintained by gabriman Hosted on GitHub Pages — Theme by mattgraham

¿Que es CameraControlInterface?

¿No sería genial poder utilizar obtener imágenes con una cámara desde Matlab para procesarlas en tiempo real? ¿Y si esta cámara fuese una réflex con la calidad fotográfica y las posibilidades de configuración que este tipo de cámaras proporcionan? Todo esto es posible con este Toolbox de Matlab que nos permite el desarrollo de aplicaciones de Matlab permitiendo interactuar con una cámara conectada por USB al ordenador.

Componentes del proyecto

El proyecto está dividido en 3 partes, siendo la tercera dividida en otros 3 partes.

  • Interfaz del programa: Programa que escribe y lee comandos de ficheros XML y ejecuta dichos comandos en las cámaras conectadas compatibles. Este programa actuará de interfaz entre el toolbox y los SDK de cada una de las cámaras. Repositorio
  • Toolbox del programa: Toolbox de Matlab que proporcione funciones al desarrollador ousuario de Matlab para que interacciones con las cámaras a través del interface anteriormente descrito. Repositorio
  • Programas de ejemplo: Creados para demostrar el funcionamiento de los programas.
    • Control básico de la cámara: Aplicación Matlab para el control de las funciones básicas de la cámara utilizando el Toolbox previamente creado. Repositorio
    • Captura de HDR: Aplicación Matlab para la creación de imágenes HDR (alto rango dinámico) utilizando el Toolbox previamente creado, tomando las tomas necesarias con la cámara conectada. Repositorio
    • Creador de timelapses: Aplicación Matlab para la creación de time-lapses utilizando el Toolbox previamente creado, tomando las imágenes necesarias con la cámara conectada. Repositorio
Para el correcto funcionamiento del programa siempre serán necesarias las 2 primeras partes, pudiendo utilizar los programas de ejemplo solo como guía de como se podrían desarrollar nuevos programas.

Características

La arquitectura con la que está diseñado el proyecto proporciona la ventaja de permitir que el toolbox sea también ejecutable en 64 bits. Esta es la ventaja principal que motiva el objetivo del proyecto: crear un toolbox de Matlab para interactuar con las cámaras réflex Nikon y Canon. Además, gracias a las modicaciones efectuadas en la arquitectura inicial, se ha dotado al proyecto de una independencia entre el control de las cámaras y el propio toolbox de Matlab, ya que solamente están unidos por las escritura/lectura de ficheros XML, por lo que se podrá hacer uso de las funciones programas en el programa interfaz desde cualquier lenguaje o programa que permita el lanzamiento de un programa externo (con una orden de Windows) para el inicio del programa y la lectura/escritura en ficheros de texto. Esto es una ventaja muy interesante, ya que realmente estamos creando un proyecto con una gran posibilidad de expansión. Por otro lado, el programa interfaz está diseñado de tal manera que es sencillo modificarlo para agregar nuevos modelos de cámara no soportados o incluso nuevas marcas, teniendo así un proyecto facilmente ampliable tanto por el lado de las camaras como por el lado del usuario (desde donde se interactuará).

Notas

Aunque el proyecto ha sido desarrollado para poder utilizarse con cámaras Canon y Nikon, las funcionalidades que soportan Nikon no están 100% funcionales por lo que no se recomienda su utilización hasta que sean solucionados los errores que se producen. Por otro lado, el funcionamiento para las cámaras Canon sí está operativo, aunque solo se puede garantizar el funcionamiento en la Canon 550D, aunque debería de funcionar sin problemas en los demás modelos de Canon soportados por el SDK 2.14 de Canon.
